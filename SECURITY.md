# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Binaris, please report it privately.

**Email:** hellospideydev@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected component or tool
- Potential impact

Do not open a public issue for security vulnerabilities. I will respond within 48 hours.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest commit | Yes |
| Older versions | No |

## Security Best Practices

- Binaris runs entirely in your browser
- No data is sent to any server
- All JWT processing is client-side
- No tracking or analytics
- No external dependencies

## Disclaimer

Binaris is a security testing tool intended for:

- Educational purposes
- Authorized security assessments
- Personal development testing

Users are responsible for complying with applicable laws. Do not use Binaris on systems you do not own or have permission to test.
