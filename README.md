## About

Binaris is a professional JWT security testing suite for developers and security researchers. Analyze, decode, and test JSON Web Tokens for common vulnerabilities.

## Tools

| Tool | Description |
|------|-------------|
| **Decode** | Decode header, payload, and signature with real-time security alerts |
| **Algorithm Analysis** | Detect algorithm type and identify misconfigurations |
| **alg=none Exploit** | Test if server accepts tokens without signature verification |
| **Weak Secret Scanner** | Test against common weak secrets and default passwords |
| **Time Claim Inspector** | Analyze expiration, not-before, and issued-at claims |
| **Key Confusion Test** | Test RS256 to HS256 algorithm confusion vulnerability |
| **Claims Deep Dive** | View all JWT claims with detailed explanations |
| **Token Forge** | Generate test JWT tokens with custom payload |

## Usage

Open `index.html` in any browser. No installation required.
